# Imverica Mobile (iOS + Android)

Capacitor wrapper around the Imverica web app. One codebase → both stores.

## Architecture

- **Native shell** (this directory): iOS + Android projects generated by Capacitor.
- **Web content**: loads live from `https://imverica.com` (see `capacitor.config.ts`).
- **Offline fallback**: minimal HTML in `www/index.html` shown if the network is
  unreachable on cold launch.
- **Native value-add** (passes App Store Review Guideline 4.2):
  - Push notifications (case status, new messages, prepared documents ready)
  - Camera document scanning (passport, court paperwork, evidence)
  - Biometric login (Face ID / Touch ID / Android biometrics)
  - Native share sheet (export prepared PDFs)
  - System-level deep linking back into the web app

## Prerequisites

Install once on this Mac:

| Tool | Purpose | Install |
|------|---------|---------|
| Xcode (full app, not just CLT) | iOS build + simulator | **App Store → "Xcode"** (~15 GB) |
| Xcode Command Line Tools | toolchain | `xcode-select --install` |
| CocoaPods | iOS dependency manager | `brew install cocoapods` ✅ done |
| Android Studio | Android build + emulator | https://developer.android.com/studio (~1 GB) |
| Java JDK 17+ | Android Gradle | bundled with Android Studio |
| Apple Developer account | App Store distribution | https://developer.apple.com — $99/yr |
| Google Play Console | Play Store distribution | https://play.google.com/console — $25 one-time |

After Xcode is installed, run once:

```bash
sudo xcode-select -s /Applications/Xcode.app/Contents/Developer
sudo xcodebuild -license accept
```

## Daily workflow

```bash
# 1. Sync after web/native changes
LANG=en_US.UTF-8 LC_ALL=en_US.UTF-8 npx cap sync

# 2. Open iOS in Xcode
LANG=en_US.UTF-8 LC_ALL=en_US.UTF-8 npx cap open ios
# → Xcode opens. Press ⌘R to run on simulator or connected device.

# 3. Open Android in Android Studio
LANG=en_US.UTF-8 LC_ALL=en_US.UTF-8 npx cap open android
# → Android Studio opens. Click ▶ Run to launch on emulator or device.
```

## Plugins installed

- `@capacitor/core` — runtime bridge
- `@capacitor/app` — app lifecycle events
- `@capacitor/browser` — open external URLs in in-app browser
- `@capacitor/camera` — take photos / scan documents
- `@capacitor/preferences` — small key-value storage
- `@capacitor/push-notifications` — APNs (iOS) + FCM (Android)
- `@capacitor/share` — system share sheet
- `@capacitor/splash-screen` — launch splash control
- `@capacitor/status-bar` — status-bar styling

## What still needs setup before App Store / Play Store

1. **App icons + splash screens** — generate from `astro-site/public/imverica-logo.svg`
   using `@capacitor/assets` (`npx capacitor-assets generate`).
2. **iOS signing**:
   - Open `ios/App/App.xcworkspace` in Xcode.
   - In *Signing & Capabilities*, select your Apple Developer team.
   - Enable *Push Notifications* and *Sign in with Apple* capabilities.
3. **Android signing**:
   - Generate a release keystore (`keytool -genkey -v ...`).
   - Add `signingConfigs` block to `android/app/build.gradle`.
   - Store keystore password in env, never in repo.
4. **APNs (iOS push)**:
   - In Apple Developer console, create an APNs Auth Key.
   - Upload to Firebase Cloud Messaging (or your push backend).
5. **FCM (Android push)**:
   - In Firebase console, create project, add Android app `com.imverica.app`.
   - Download `google-services.json` into `android/app/`.
6. **TestFlight + internal Play track** before public release.

## Updating native projects

When you upgrade Capacitor or add a plugin:

```bash
# Bump Capacitor versions in package.json, then:
npm install
LANG=en_US.UTF-8 LC_ALL=en_US.UTF-8 npx cap sync
```

If iOS `pod install` fails, run manually:

```bash
cd ios/App
LANG=en_US.UTF-8 LC_ALL=en_US.UTF-8 pod install
```

## Troubleshooting

- **`pod install` Unicode error** → always export `LANG=en_US.UTF-8 LC_ALL=en_US.UTF-8`
  before any CocoaPods command.
- **`xcodebuild: tool requires Xcode`** → you only have Command Line Tools.
  Install full Xcode from the App Store, then run `sudo xcode-select -s
  /Applications/Xcode.app/Contents/Developer`.
- **Capacitor CLI complains about Node version** → we pin Capacitor v7
  (works with Node ≥18). Don't bump to v8 unless we also upgrade Node to ≥22.
